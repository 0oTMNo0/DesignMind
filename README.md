## 🚀 How to use

> `npx create-expo my-app --example with-reanimated`

- Run `yarn` or `npm install`
- Run `yarn start` or `npm run start` to try it out.

- To run your project, run one of the following npm commands.

- npm run android
- npm run ios
- npm run web
